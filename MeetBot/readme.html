How to Configure

Open the extension and click the day that you would like to configure. Enter in your links and times, you can either save each slot individually with the save button to the right of the slot, or you can save them all with the save all button. However, this also saves the unconfigured slots, so only do this when you want to configure the whole day.


How to Use

Now that the day is configured, you can let the extension run. It will check the time every two minutes, so it will join your meet either two or three minutes before the start time. Make sure to leave browser open at least five minutes before the meet to allow it to actually join.


Troubleshooting

If your extension isn't joining a meet, check the configuration of that day using the configuration button. If that is working, go to the extension page right after the extension fails, and click the background page hyperlink in the extension box. Read what the console says. If there isn't any message indicating an error, or if a portion of the configuration page isn't working properly, submit a bug report on the GitHub page or send me an email.
